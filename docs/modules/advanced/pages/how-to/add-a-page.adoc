= How to Add a Page

This is a practical how-to.

== 1. Create the page file

Put your file here:

[source,text]
----
docs/modules/<module>/pages/<path>/my-page.adoc
----

== 2. Add an xref somewhere

Example:

`xref:some-page.adoc[Some Page]`

== 3. Add it to navigation

Edit `docs/modules/ROOT/nav.adoc` and add:

`* xref:<module>:<path>/my-page.adoc[My Page]`

== 4. Build

[source,bash]
----
npm run build
----
