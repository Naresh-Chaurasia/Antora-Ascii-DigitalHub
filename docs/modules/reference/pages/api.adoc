= API Reference

This module provides comprehensive API documentation.

== Core API

=== Project API

==== Create Project

[source,http]
----
POST /api/projects
Content-Type: application/json

{
  "name": "My Project",
  "description": "Project description",
  "settings": {
    "theme": "default",
    "public": true
  }
}
----

Response:
[source,json]
----
{
  "id": "proj_123",
  "name": "My Project",
  "created_at": "2023-01-01T00:00:00Z",
  "url": "https://api.example.com/projects/proj_123"
}
----

==== Get Project

[source,http]
----
GET /api/projects/{id}
Authorization: Bearer {token}
----

Response:
[source,json]
----
{
  "id": "proj_123",
  "name": "My Project",
  "description": "Project description",
  "settings": {
    "theme": "default",
    "public": true
  },
  "created_at": "2023-01-01T00:00:00Z",
  "updated_at": "2023-01-02T00:00:00Z"
}
----

==== Update Project

[source,http]
----
PUT /api/projects/{id}
Authorization: Bearer {token}
Content-Type: application/json

{
  "name": "Updated Project",
  "description": "Updated description"
}
----

==== Delete Project

[source,http]
----
DELETE /api/projects/{id}
Authorization: Bearer {token}
----

=== Build API

==== Trigger Build

[source,http]
----
POST /api/projects/{id}/builds
Authorization: Bearer {token}

{
  "branch": "main",
  "environment": "production"
}
----

Response:
[source,json]
----
{
  "id": "build_456",
  "project_id": "proj_123",
  "status": "pending",
  "created_at": "2023-01-01T00:00:00Z"
}
----

==== Get Build Status

[source,http]
----
GET /api/builds/{id}
Authorization: Bearer {token}
----

Response:
[source,json]
----
{
  "id": "build_456",
  "project_id": "proj_123",
  "status": "success",
  "started_at": "2023-01-01T00:01:00Z",
  "completed_at": "2023-01-01T00:05:00Z",
  "logs": [
    "Build started...",
    "Installing dependencies...",
    "Building project...",
    "Build completed successfully!"
  ]
}
----

== Authentication

=== API Key Authentication

Generate API keys in the dashboard:

[source,bash]
----
curl -H "X-API-Key: your-api-key" \
     https://api.example.com/projects
----

=== OAuth2 Authentication

Get access token:

[source,http]
----
POST /oauth/token
Content-Type: application/x-www-form-urlencoded

grant_type=client_credentials&client_id=YOUR_ID&client_secret=YOUR_SECRET
----

Use token:

[source,bash]
----
curl -H "Authorization: Bearer ACCESS_TOKEN" \
     https://api.example.com/projects
----

== Error Handling

=== Error Response Format

All errors return consistent format:

[source,json]
----
{
  "error": {
    "code": "NOT_FOUND",
    "message": "Project not found",
    "details": {
      "project_id": "invalid_id"
    }
  }
}
----

=== Common Error Codes

| Code | Description | HTTP Status |
|------|-------------|------------|
| `NOT_FOUND` | Resource not found | 404 |
| `UNAUTHORIZED` | Invalid authentication | 401 |
| `FORBIDDEN` | Insufficient permissions | 403 |
| `VALIDATION_ERROR` | Invalid input data | 400 |
| `RATE_LIMITED` | Too many requests | 429 |
| `INTERNAL_ERROR` | Server error | 500 |

=== Rate Limiting

API requests are rate limited:

| Plan | Requests per hour |
|------|-------------------|
| Free | 100 |
| Pro | 1000 |
| Enterprise | 10000 |

Rate limit headers:

[source,http]
----
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1640995200
----

== SDKs and Libraries

=== JavaScript/Node.js

[source,javascript]
----
const { Client } = require('@ourtool/sdk');

const client = new Client({
  apiKey: 'your-api-key',
  baseURL: 'https://api.example.com'
});

// Create project
const project = await client.projects.create({
  name: 'My Project',
  description: 'Project description'
});

// Trigger build
const build = await client.builds.trigger(project.id, {
  branch: 'main'
});
----

=== Python

[source,python]
----
from ourtool_sdk import Client

client = Client(api_key='your-api-key')

# Create project
project = client.projects.create({
    'name': 'My Project',
    'description': 'Project description'
})

# Trigger build
build = client.builds.trigger(project.id, branch='main')
----

=== Go

[source,go]
----
package main

import (
    "github.com/ourtool/go-sdk"
    "context"
)

func main() {
    client := sdk.NewClient("your-api-key")
    
    // Create project
    project, err := client.Projects.Create(context.Background(), &sdk.Project{
        Name: "My Project",
        Description: "Project description",
    })
    
    // Trigger build
    build, err := client.Builds.Trigger(context.Background(), project.ID, "main")
}
----

== Webhooks

=== Configure Webhooks

Set up webhooks to receive notifications:

[source,json]
----
{
  "url": "https://your-site.com/webhook",
  "events": ["build.started", "build.completed", "build.failed"],
  "secret": "webhook-secret"
}
----

=== Webhook Payload

[source,json]
----
{
  "event": "build.completed",
  "data": {
    "build": {
      "id": "build_456",
      "project_id": "proj_123",
      "status": "success",
      "completed_at": "2023-01-01T00:05:00Z"
    }
  },
  "timestamp": "2023-01-01T00:05:00Z"
}
----

=== Verify Webhook

Verify webhook signature:

[source,javascript]
----
const crypto = require('crypto');

function verifyWebhook(payload, signature, secret) {
  const hmac = crypto.createHmac('sha256', secret);
  hmac.update(payload);
  const expectedSignature = hmac.digest('hex');
  return signature === expectedSignature;
}
----
