== Package Managers

=== npm

[source,bash]
----
# Latest stable
npm install -g @ourtool/cli

# Development version
npm install -g @ourtool/cli@next

# Specific version
npm install -g @ourtool/cli@1.2.3
----

=== Homebrew (macOS)

[source,bash]
----
# Stable version
brew install ourtool

# Development version
brew install ourtool --HEAD

# Update
brew upgrade ourtool
----

=== Chocolatey (Windows)

[source,powershell]
----
# Stable version
choco install ourtool

# Development version
choco install ourtool --pre

# Update
choco upgrade ourtool
----

=== Scoop (Windows)

[source,powershell]
----
# Add bucket
scoop bucket add ourtool https://github.com/ourtool/scoop

# Install
scoop install ourtool

# Update
scoop update ourtool
----

=== APT (Ubuntu/Debian)

[source,bash]
----
# Add repository
curl -fsSL https://packages.example.com/apt/gpg | sudo apt-key add -
echo "deb https://packages.example.com/apt stable main" | sudo tee /etc/apt/sources.list.d/ourtool.list

# Install
sudo apt update
sudo apt install ourtool

# Update
sudo apt update && sudo apt upgrade ourtool
----

=== YUM/DNF (RHEL/CentOS/Fedora)

[source,bash]
----
# Add repository
sudo rpm --import https://packages.example.com/rpm/gpg
sudo tee /etc/yum.repos.d/ourtool.repo > /dev/null <<EOF
[ourtool]
name=OurTool Repository
baseurl=https://packages.example.com/rpm/stable/\$basearch
gpgcheck=1
gpgkey=https://packages.example.com/rpm/gpg
EOF

# Install
sudo yum install ourtool

# Update
sudo yum update ourtool
----

=== Docker

[source,bash]
----
# Pull image
docker pull ourtool/cli:latest

# Run container
docker run -it ourtool/cli:latest

# Mount current directory
docker run -it -v $(pwd):/workspace ourtool/cli:latest
----

=== Docker Compose

[source,yaml]
----
version: '3.8'
services:
  ourtool:
    image: ourtool/cli:latest
    volumes:
      - .:/workspace
    working_dir: /workspace
    command: serve --port 3000 --host 0.0.0.0
    ports:
      - "3000:3000"
----

== Verification

After downloading, verify the integrity of the file:

=== Verify Checksum

[source,bash]
----
# Calculate SHA256
sha256sum ourtool-linux-x64.tar.gz

# Compare with expected
echo "abc123... ourtool-linux-x64.tar.gz" | sha256sum -c -
----

=== Verify Signature

[source,bash]
----
# Import GPG key
gpg --import ourtool-public-key.asc

# Verify signature
gpg --verify ourtool-linux-x64.tar.gz.sig ourtool-linux-x64.tar.gz
----

== Release Channels

=== Stable

Stable releases are recommended for production use:

* Regular release schedule (every 6 weeks)
* Thoroughly tested
* Long-term support (12 months)
* Backported security fixes

=== Beta

Beta releases for early adopters:

* Released every 2 weeks
* Feature-complete but may have bugs
* No long-term support
* Feedback encouraged

=== Alpha

Alpha releases for developers and testers:

* Released as needed
* Experimental features
* May be unstable
* Not for production use
