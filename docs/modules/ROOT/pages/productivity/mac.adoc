=== Finder

* Hide/Show Hidden Files
** Press `Command + Shift + .`

---

*Open a Folder Using Path on macOS*

*Method 1: Finder*

1. Open Finder
2. Press `⌘ + Shift + G`
3. Enter the folder path
4. Press `Enter`

Example:
----
/Users/nareshchaurasia/Documents/project
----

*Method 2: Terminal*

Open folder in Finder:

[source,bash]
----
open /Users/nareshchaurasia/Documents/project
----

Navigate in terminal:

[source,bash]
----
cd /Users/nareshchaurasia/Documents/project
----

* [ ] What is Annotate?
* [ ] Annotate means adding visual markings on your screenshot to explain something
* [ ] Examples:
* [ ] Draw arrows → point to important areas
* [ ] Add text → explain what is happening
* [ ] Highlight → focus attention
* [ ] Blur / pixelate → hide sensitive data (passwords, emails)
* [ ] Draw boxes → group sections

* [ ] How to Annotate in Shottr
* [ ] Take screenshot (Cmd + Shift + 2)
* [ ] Shottr editor opens automatically
* [ ] Use keyboard shortcuts:
* [ ] A → Arrow
* [ ] T → Text
* [ ] R → Rectangle
* [ ] B → Blur
* [ ] Make quick edits and press S to save

* [ ] How to Save Directly to docs/modules/ROOT/assets/images/
* [ ] Open Shottr Settings
* [ ] Go to → Saving / File Settings
* [ ] Set Default Save Location:
* [ ] Select your Antora folder:
* [ ] docs/modules/ROOT/assets/images/

* [ ] Optional (Recommended)
* [ ] Enable auto-save after capture
* [ ] Enable auto-copy to clipboard
* [ ] Set file naming format (e.g., screenshot-{date}-{time}.png)

* [ ] Result
* [ ] Every screenshot you take will be saved automatically in your Antora images folder
* [ ] No manual move needed

=== Safari

* *Basic navigation*
** Swipe left or right with two fingers
** Or use Command + Left / Right Arrow

* *Tab management*
* Open new tab:
** Click "+" icon
** Or press Command + T
* Switch tabs:
** Press Control + Tab repeatedly
* Close tab:
** Press Command + W
* Reopen closed tab:
** Press Command + Z

* *Closing multiple tabs*

* Close all other tabs:

* Press Command + Option + W
* Note:

* Command + Z will NOT restore closed tabs in this case

* *Managing many tabs*
* View all tabs:
** Pinch inward on trackpad
** Or press Command + Shift + Backslash ()
* Exit tab overview:
** Press Command + Shift + Backslash again

* *Refreshing pages*
* Reload page:
** Click reload button
** Or press Command + R

* *Scrolling alternatives*
** Use Option + Up / Down Arrow
** Useful when mouse is used instead of trackpad

* *Finding text on a page*

* Open search:

** Press Command + F
** Type keyword (e.g., "Japan")
** Matches are highlighted on page
** Scroll to navigate through results

* *Closing browser*

* Close current tab:

** Command + W
* Quit browser completely:
    
* Command + Q
