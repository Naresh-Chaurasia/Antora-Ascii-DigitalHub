*MCP (Model Context Protocol)* allows AI models to connect with external tools like filesystems, databases, and GitHub repositories.

With MCP, an AI assistant inside your editor can *read repositories, create issues, analyze code, and automate workflows.*

== Basic Idea

Architecture:

----

VS Code
↓
AI Agent / Extension
↓
MCP Client
↓
MCP Server (GitHub / Filesystem / Database)
↓
External Tool

----

Example:VS Code AI → MCP GitHub Server → Your GitHub Repo

== Steps

*Step 1 – Install an MCP-capable AI Extension in VS Code*

Extensions that support MCP include:

* Claude Desktop with VS Code integration
* Continue.dev extension
* Cursor IDE

A simple option is installing the Continue extension inside VS Code.

##############################################

*Step 2 – Install MCP Server for GitHub*

MCP servers are typically distributed as Node.js packages.

Example GitHub MCP server:


`npx -y @modelcontextprotocol/server-github`

This server allows AI to:

* Read repositories
* Search code
* Create issues
* Read pull requests
* Create pull requests

##############################################

*Step 3 – Configure MCP in VS Code*

In your AI extension configuration file, define the MCP server.

Example configuration:


----

{
"mcpServers": {
  "github": {
      "command": "npx",
      "args": [
      "-y",
      "@modelcontextprotocol/server-github"
      ],
      "env": {
      "GITHUB_TOKEN": "your_token_here"
      }
      }
  }
}
----

After this configuration, the AI can communicate with GitHub through MCP.

*Step 4 – Create GitHub Token*

Go to:

* GitHub → Settings → Developer Settings → Personal Access Token.

Give permissions such as:

* repo
* read:org
* workflow

Copy the generated token and add it to the MCP configuration.

*Example Prompts You Can Run in VS Code*

Once MCP is connected, you can ask the AI assistant:

* List files in my repository
* Find all Spring Boot controllers in this repo
* Create an issue called "Bug in login API"
* Explain this repository architecture

The AI will actually read and interact with your repository.
