== Learning Strategy

* Watch Overview
* Do Quickstart hands-on
* Learn Core Concepts deeply
* Apply on your own docs
* Master AsciiDoc
* Move to UI + Extensions

TIP: Don’t binge-watch. Build while learning.

== Goal (For You)

* Create a multi-repo documentation portal
* Version your courses like software
* Reuse content using includes
* Automate build using CI/CD
* Build your personal documentation platform

TIP: Think of Antora as "Documentation Architecture", not just a tool.
