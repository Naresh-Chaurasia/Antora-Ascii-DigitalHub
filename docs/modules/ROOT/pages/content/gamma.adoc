=== ✔️ How to Add and Edit Images in Gamma

video::5CrK7ytrRJk[youtube, width=600, height=350, caption="How to Add and Edit Images in Gamma", align="left"]

.How to Add and Edit Images in Gamma
[%collapsible]
====

* [x] #Upload# images from your device (photos, logos, branded visuals)
* [x] Paste an image #URL# to import images instantly
* [x] Use #built-in web search# to find images without leaving Gamma
* [x] Filter web images by license type
* [ ] Generate #AI images# by describing what you want
* [ ] Choose style, aspect ratio, and AI model for generated images
* [ ] Access free, high-quality stock photos from #Unsplash# within Gamma
* [ ] Add #GIFs# from Gamma’s built-in GIF library to make content dynamic
* [ ] Use #pictographic#, hand-drawn style illustrations for a creative touch
* [ ] Search and add #icons# to emphasize points or explain concepts
* [ ] Use #accent images# to match your theme and fill design spaces
* [ ] ❌ Manage accent images directly from the theme editor
* [ ] Generate and insert #QR codes# to link to external resources
* [ ] Add images while generating a new Gamma by selecting image sources in the prompt editor
* [ ] Gamma automatically suggests and places relevant visuals during creation
* [ ] Add images to an existing Gamma using the image icon in the insert menu
* [ ] Drag and drop images directly into cards
* [ ] Crop images to focus on important areas
* [ ] Adjust image fit for better layout inside cards
* [ ] Set focus points so images look good on all devices
* [ ] Apply predefined styles to AI images (realistic, artistic, or custom)
* [ ] Use AI-powered editing features like background removal, upscaling, vectorizing, and animation
* [ ] Modify images using natural language (change colors, add/remove elements, refine details)
* [ ] Gamma enables quick, flexible image editing without advanced design skills
* [ ] Users can start for free at gamma.app to create visually rich content
