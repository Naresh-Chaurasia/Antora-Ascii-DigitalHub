== Overview

This guide walks you step-by-step to build a **Text-to-Speech system using your own cloned voice**.

You will:

* Record your voice dataset
* Prepare training data
* Train a voice model using Coqui TTS
* Generate speech from text
* Build a simple API around it

---

== Prerequisites

*Basic Requirements*

* Python 3.9 – 3.11
* Mac / Linux (Mac M1/M2 works)
* 16GB RAM (recommended)
* GPU (optional but highly recommended for faster training)

##############################################

---

== High-Level Architecture

* Text Input → Voice Model → Audio Output
* Extended pipeline: Dataset → Preprocessing → Training → Model → Inference → API → Video

== Step 1: Record Your Voice Dataset

This is the *most critical step*.

*Recording Guidelines*

* Quiet room (no fan, AC noise)
* Same microphone throughout
* Speak naturally (teaching tone)
* Avoid background noise
* Maintain consistent distance from mic

*Duration*

* Minimum: 30 minutes
* Recommended: 60–90 minutes

---

*What to Record*

Include variety:

* Statements
* Questions
* Explanations
* Emotional variations

*File Format*

* Format: WAV
* Bit depth: 16-bit
* Sample rate: 22050 Hz or 44100 Hz

---

*File Naming*

```
audio_001.wav
audio_002.wav
audio_003.wav
```

== Step 2: Create Metadata File

Create a file named `metadata.txt`:

Format:

```
audio_001|Hello everyone, welcome to my Java course
audio_002|Today we will learn inheritance
audio_003|What is polymorphism
```

*Important Rules*

* Exact match between audio and text
* No spelling mistakes
* Use simple punctuation

== Step 6: Choose Model (Recommended)

Use pretrained multilingual model:

tts_models/multilingual/multi-dataset/xtts_v2

Why:

* Supports voice cloning
* Good quality
* Less training time

---

##############################################

---

== Step 7: Training / Fine-Tuning

=== Basic Command

```bash
tts --model_name tts_models/multilingual/multi-dataset/xtts_v2 \
--data_path ./dataset \
--output_path ./output
```

---

=== What Happens

* Loads pretrained model
* Learns your voice patterns
* Saves trained model

---

=== Training Time

* CPU: Very slow (not recommended)
* GPU: 2–10 hours depending on dataset

---

== Step 9: Generate Speech (Inference)

```bash
tts --text "Welcome to my course" \
--model_path ./output/model.pth \
--config_path ./output/config.json \
--out_path output.wav
```

== Step 10: Build API (Optional but Recommended)

=== Python FastAPI Example

```python
from fastapi import FastAPI
import os

app = FastAPI()

@app.get("/speak")
def speak(text: str):
    os.system(f'tts --text "{text}" --model_path ./output/model.pth --out_path output.wav')
    return {"status": "generated"}
```

Run:

```bash
uvicorn main:app --reload
```
