= Learn CapCut Efficiently
:toc: right

== First Principle

> Don't "Complete the Course" — Learn the Skill

Your goal is *not* to finish all 51 lessons. Your *goal is*: Be able to create Instagram videos from your dance footage confidently. Everything else is secondary.

== Why It Feels Overwhelming

This kind of course mixes:

* Passive learning (watching)
* Active doing (editing along)

Doing both for every lesson leads to:

* 3–4x time increase
* Mental fatigue
* Burnout

== Best Strategy (High ROI Method)

=== 1. Use the 3-Pass Approach

*Pass 1 – Fast Overview (No Doing)*

* Watch at 1.25x–1.5x speed
* Don’t pause
* Don’t follow along
* Focus on understanding:
** What CapCut can do
** What features exist

Time required:

* 2–3 hours instead of 10+

##############################################

---

*Pass 2 – Targeted Practice (Selective Doing)*

Focus only on what you need for Instagram dance videos:

* Importing footage
* Timeline editing (cut, trim, split)
* B-roll (dance clips)
* Text and captions
* Transitions
* Music sync
* Export

Skip for now:
* Green screen
* Teeth whitening
* Multicam editing
* Proxy editing

##############################################

---

*Pass 3 – Real Project (Most Important)*

Use your own:
* Bachata practice videos
* Partner clips
* Group clips

Create:
* 1 Instagram reel (30–60 seconds)

When stuck:
* Go back to specific lessons
* Learn only what is needed

##############################################

---

== 2. Apply the 70-20-10 Rule

* 10% → Watching the course
* 20% → Guided practice
* 70% → Your own projects

Avoid reversing this ratio.

##############################################

---

== 3. Don’t Follow Along Line-by-Line

Instead of:
* Pause → replicate → resume

Do:
* Watch the full lesson
* Then recreate from memory

Benefit:
* Builds real skill instead of dependency

##############################################

---

== 4. Create a Minimal Skill Checklist

For dance reels, you only need:

* Cut and trim clips
* Sync with music
* Add text (song name, vibe)
* Add transitions
* Basic color correction
* Export in 9:16 format

This is enough to start posting.

##############################################

---

== 5. Time-Box Your Learning

* Limit sessions to 45–60 minutes
* Stop even if incomplete

Focus on:
* Consistency over intensity

##############################################

---

== 6. Use Just-in-Time Learning

When stuck:

* Go to the specific lesson
* Learn the feature
* Apply immediately

This mirrors how professionals learn tools.

##############################################

---

== 7. Motivation Hack (For Dance Content)

Turn learning into a habit:

* Every practice session → create 1 reel

Outcome:
* Learning becomes part of your lifestyle

##############################################

---

== Simple Plan

=== Day 1

* Watch lessons 1–12 quickly (no practice)

=== Day 2

* Watch lessons 12–20
* Do basic practice

=== Day 3

* Create your first reel using your own dance footage

=== Ongoing

Improve one aspect per video:

* Captions
* Transitions
* Effects

== Big Mindset Shift

You do not need:

* Full course completion
* Perfect understanding
* Every feature

You only need:
* Good enough skills to create and post content
