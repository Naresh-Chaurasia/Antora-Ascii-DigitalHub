= Antora Terminology

This page is in a subfolder (`concepts/`) to show how you can organize pages.

== Core terms

* *Component* - A versioned set of documentation (configured in `docs/antora.yml`).
* *Module* - A logical grouping inside a component (`modules/<name>/`).
* *Page* - An AsciiDoc file that becomes an HTML page.
* *Nav* - The sidebar navigation defined in `nav.adoc`.
